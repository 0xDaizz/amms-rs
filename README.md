# Dark Automated Market Makers
A Rust library to interact with automated market makers across EVM chains.

To launch the docs, run the following command in your terminal. 
```
cargo doc --open
```

## Supported Dexes

| Dex | Status |
|----------|------|
| UniswapV2 variants  | ✅||
| UniswapV3  | ✅||


## Running Examples

To run any of the examples, first set a local environment variable called `MAINNET_RPC_ENDPOINT`. Then you can simply run `cargo run --example <example_name>`.
